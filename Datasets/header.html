The COSMOS2020 catalogue Release 1 | v2.2 | March 2023

This is the release site for the COSMOS2020 catalogs. In addition to this website, the catalogues are now being made available through ESO's PHASE3 system and later through CDS. Catalogues are described in detail in Weaver, Kauffmann et al. 2022, ApJS. A local version is here. This release supersedes previous releases. The principal difference compared to our previous release is that this new release is fully IVOA-compliant. This document describes the changes in detail. As of March 2023, we have patched a minor bug in the magnitude error values in the v2.1 Farmer catalogue. The Classic catalog is not affected, but has been renamed to v2.2 for consistency. We also provide a python notebook demonstrating how to use the catalogues.

Secondary access site

A compressed tar file containing all the files here can be downloaded here (8.6Gb) (note that this does not include the FITS detection image). Alternatively files individually from this page.

Construction of the catalogues

We derived the photometry with several software packages: Source Extractor with IRACLEAN, and The Farmer which is built on The Tractor and will be described in a forthcoming paper. For each catalogue, photometric and physical parameters are provided by Le Phare and EAzY.

Catalogue descriptions

A detailed description of these data products is provided in the catalogue release document, and supplementary document regarding the recent bug fix (see Sect. 2.3 for details). Here, we simply list each file with a brief description.

The contents of each redshift distribution file can be found in their respective file headers.
